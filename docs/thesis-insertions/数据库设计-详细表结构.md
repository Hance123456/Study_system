## 5.3 数据库设计

本系统采用MySQL作为数据库管理系统，设计了10个核心数据表，涵盖用户管理、知识卡片、学习进度、复习计划、测验记录和统计分析等功能模块。数据库设计充分考虑了间隔重复学习（Spaced Repetition System, SRS）的业务需求，重点优化了复习计划生成、学习进度跟踪和统计分析的查询性能。

### 表5-1 系统核心数据表清单

| 序号 | 表名              | 中文名称       | 主要功能说明                          | 关键字段 |
|------|-------------------|----------------|---------------------------------------|----------|
| 1    | users            | 用户表         | 存储微信小程序用户基本信息             | openid, nickname, status |
| 2    | admins           | 管理员表       | 后台管理系统管理员账号                 | username, role, status |
| 3    | courses          | 课程分类表     | 知识卡片分类管理                       | name, sort_order, status |
| 4    | cards            | 知识卡片表     | 核心学习内容载体，支持富文本和音频     | title, content, audio_url, difficulty |
| 5    | user_progress    | 用户学习进度表 | 记录用户对每个卡片的掌握程度和复习计划 | mastery_level, next_review_at, review_count |
| 6    | review_plans     | 复习计划表     | 基于艾宾浩斯曲线的每日复习任务调度     | plan_date, review_stage, is_completed |
| 7    | quizzes          | 测验题目表     | 卡片关联的测验题目，支持多种题型       | question, question_type, options, answer |
| 8    | quiz_records     | 测验记录表     | 用户答题记录，用于学习效果分析         | is_correct, time_spent |
| 9    | learning_logs    | 学习日志表     | 记录用户学习行为（阅读、听音频、复习、测验） | action_type, duration, learning_mode |
| 10   | user_daily_stats | 用户每日统计表 | 按日聚合学习数据，用于打卡日历和学习报表 | study_duration, cards_learned, is_checked_in |

### 表5-2 users 用户表

| 字段名        | 类型           | 可空 | 默认值   | 说明                           | 备注 |
|---------------|----------------|------|----------|--------------------------------|------|
| id            | INT            | NO   | AUTO     | 主键                           | - |
| openid        | VARCHAR(100)   | NO   | -        | 微信openid（唯一）             | 唯一索引 |
| union_id      | VARCHAR(100)   | YES  | NULL     | 微信unionid                    | - |
| nickname      | VARCHAR(50)    | YES  | NULL     | 用户昵称                       | - |
| avatar        | VARCHAR(500)   | YES  | NULL     | 头像URL                        | - |
| gender        | TINYINT        | YES  | 0        | 性别（0未知、1男、2女）         | - |
| phone         | VARCHAR(20)    | YES  | NULL     | 手机号                         | - |
| created_at    | DATETIME       | YES  | NOW()    | 注册时间                       | - |
| updated_at    | DATETIME       | YES  | NOW()    | 更新时间                       | - |
| last_login_at | DATETIME       | YES  | NULL     | 最后登录时间                   | - |
| status        | TINYINT        | YES  | 1        | 状态（0禁用、1正常）           | - |

**索引**：`idx_openid(openid)`

---

### 表5-3 cards 知识卡片表（核心业务表）

| 字段名      | 类型           | 可空 | 默认值   | 说明                           | 备注 |
|-------------|----------------|------|----------|--------------------------------|------|
| id          | INT            | NO   | AUTO     | 主键                           | - |
| course_id   | INT            | NO   | -        | 所属课程ID（外键）             | 索引 |
| title       | VARCHAR(200)   | NO   | -        | 卡片标题                       | - |
| content     | TEXT           | NO   | -        | 卡片内容（富文本）             | - |
| summary     | VARCHAR(500)   | YES  | NULL     | 内容摘要                       | - |
| image       | VARCHAR(500)   | YES  | NULL     | 配图URL                        | - |
| audio_url   | VARCHAR(500)   | YES  | NULL     | TTS生成的音频URL               | 核心特性 |
| difficulty  | TINYINT        | YES  | 1        | 难度等级（1简单、2中等、3困难） | - |
| sort_order  | INT            | YES  | 0        | 排序字段                       | 索引 |
| view_count  | INT            | YES  | 0        | 浏览次数                       | - |
| created_at  | DATETIME       | YES  | NOW()    | 创建时间                       | - |
| updated_at  | DATETIME       | YES  | NOW()    | 更新时间                       | - |
| status      | TINYINT        | YES  | 1        | 状态（0禁用、1正常）           | - |

**索引**：`idx_course_id(course_id)`  
**外键**：`course_id → courses(id) ON DELETE CASCADE`

---

### 表5-4 user_progress 用户学习进度表（SRS核心表）

| 字段名         | 类型         | 可空 | 默认值 | 说明                           | 备注 |
|----------------|--------------|------|--------|--------------------------------|------|
| id             | INT          | NO   | AUTO   | 主键                           | - |
| user_id        | INT          | NO   | -      | 用户ID（外键）                 | 索引 |
| card_id        | INT          | NO   | -      | 卡片ID（外键）                 | 索引 |
| mastery_level  | TINYINT      | YES  | 0      | 掌握程度（0-4）                | 核心字段 |
| review_count   | INT          | YES  | 0      | 复习次数                       | - |
| correct_count  | INT          | YES  | 0      | 答对次数                       | - |
| wrong_count    | INT          | YES  | 0      | 答错次数                       | - |
| last_study_at  | DATETIME     | YES  | NULL   | 最后学习时间                   | - |
| next_review_at | DATETIME     | YES  | NULL   | 下次复习时间（艾宾浩斯算法）   | **核心字段** |
| created_at     | DATETIME     | YES  | NOW()  | 创建时间                       | - |
| updated_at     | DATETIME     | YES  | NOW()  | 更新时间                       | - |

**唯一索引**：`uk_user_card(user_id, card_id)`  
**索引**：`idx_next_review(next_review_at)`  
**外键**：`user_id → users(id)`, `card_id → cards(id)`

---

### 表5-5 review_plans 复习计划表

| 字段名        | 类型       | 可空 | 默认值 | 说明                           | 备注 |
|---------------|------------|------|--------|--------------------------------|------|
| id            | INT        | NO   | AUTO   | 主键                           | - |
| user_id       | INT        | NO   | -      | 用户ID（外键）                 | 索引 |
| card_id       | INT        | NO   | -      | 卡片ID（外键）                 | 索引 |
| plan_date     | DATE       | NO   | -      | 计划复习日期                   | 核心字段 |
| review_stage  | TINYINT    | YES  | 1      | 复习阶段（1-6）                | 艾宾浩斯阶段 |
| is_completed  | TINYINT    | YES  | 0      | 是否完成（0/1）                | - |
| completed_at  | DATETIME   | YES  | NULL   | 完成时间                       | - |
| created_at    | DATETIME   | YES  | NOW()  | 创建时间                       | - |

**索引**：`idx_user_date(user_id, plan_date)`, `idx_plan_date(plan_date)`  
**外键**：`user_id → users(id)`, `card_id → cards(id)`

---

### 表5-6 quizzes 测验题目表

| 字段名        | 类型           | 可空 | 默认值 | 说明                           | 备注 |
|---------------|----------------|------|--------|--------------------------------|------|
| id            | INT            | NO   | AUTO   | 主键                           | - |
| card_id       | INT            | NO   | -      | 关联卡片ID（外键）             | 索引 |
| question      | TEXT           | NO   | -      | 题目内容                       | - |
| question_type | TINYINT        | YES  | 1      | 题型（1单选、2多选、3判断、4填空） | - |
| options       | JSON           | YES  | NULL   | 选项（JSON数组）               | - |
| answer        | VARCHAR(200)   | NO   | -      | 正确答案                       | - |
| explanation   | TEXT           | YES  | NULL   | 答案解析                       | - |
| sort_order    | INT            | YES  | 0      | 排序                           | - |
| created_at    | DATETIME       | YES  | NOW()  | 创建时间                       | - |
| updated_at    | DATETIME       | YES  | NOW()  | 更新时间                       | - |
| status        | TINYINT        | YES  | 1      | 状态                           | - |

**索引**：`idx_card_id(card_id)`  
**外键**：`card_id → cards(id) ON DELETE CASCADE`
