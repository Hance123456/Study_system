### 3.7 语音合成（TTS）技术

卡片音频除支持管理员手动上传外，系统还提供一键语音合成功能。该功能不依赖第三方商业语音 API 密钥，而是在服务端通过 **Python 的 edge-tts 库**调用 **微软 Edge 浏览器同源神经网络语音服务**，将文本合成为 **MP3 文件**。默认中文音色可采用 **zh-CN-XiaoxiaoNeural**，并支持通过参数调整语速（如 rate）。合成文件写入配置项 **UPLOAD_BASE_DIR** 下的 **`audio/tts`** 目录，由后端生成唯一文件名并返回可访问的相对 URL。运行环境需安装 Python 与 edge-tts 模块（`pip install edge-tts`），否则语音合成接口将无法正常执行。该方案的优点是接入成本较低、音质较好；缺点是依赖服务器出站网络与本地 Python 环境，部署时需将上述依赖纳入运维说明。

**正文（后端，可直接粘贴）：**

后端在 `upload` 路由中提供 **`POST /tts`**（与全局 API 前缀组合后为实际请求路径）。处理逻辑为：从请求体读取 **`text`、`voice`、`rate`**，对文本做长度限制；在 **`audio/tts`** 目录下生成 **`tts_时间戳_随机.mp3`**；通过 **`child_process.spawn`** 调用 **`python -m edge_tts`**（失败时回退 **`py -m edge_tts`**），参数包括 **`--voice`、`--rate`、`--text`、`--write-media`**；成功后通过既有工具函数生成相对 URL 并返回 JSON。异常场景包括未安装 Python、未安装 **edge_tts** 模块、文本为空或超长、合成进程非零退出等，统一返回错误信息供管理端提示。

**正文（管理端，可直接粘贴）：**

卡片编辑页提供 **「一键生成语音」** 按钮：从表单读取 **标题** 与 **内容**（可按设计用停顿符拼接），调用 **`/upload/tts`**，默认传入 **`zh-CN-XiaoxiaoNeural`** 与 **`+0%`** 语速。接口返回 **`data.url`** 后写入 **`audio_url`** 字段；若处于编辑态，可进一步调用持久化逻辑将 **`audio_url`** 立即保存至后端，减少用户忘记点击保存导致的数据不一致。
